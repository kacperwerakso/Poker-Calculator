# Poker Odds Calculator
An interactive web application that calculates winning probabilities for any poker hand in real time.

# Features
Poker Odds Calculator allows users to manually select two player cards and up to five community cards, then computes win probabilities using a Monte Carlo simulation with 10,000 iterations combined with full hand evaluation. The application supports simulations against multiple opponents, displays win/draw/loss percentages, detects hand potential trends such as strong flush or straight possibilities, and features a dynamic interface with live counters for an intuitive user experience.
